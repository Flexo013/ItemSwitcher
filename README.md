ItemSwitcher
============

Bukkit plugin to auto switch between tools (axe, pickaxe and shovel) as needed
when mining.  Compatible and tested with [CraftBukkit](https://github.com/Bukkit/CraftBukkit) 1185 and MC 1.8.1.
